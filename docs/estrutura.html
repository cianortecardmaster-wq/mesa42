Estrutura

A Home lista postagens. Cada postagem aponta para páginas independentes: Visual Novel, Texto, Graphic Novel ou outros formatos.

Adicionar nova postagem

Crie uma pasta em postagens/ e registre a postagem em data/posts.json. A Home vai ordenar automaticamente pela data.

{
  "date": "2026-06-25",
  "number": "História #002",
  "title": "Nome da História",
  "format": "Visual Novel",
  "url": "postagens/002-nome-da-historia/",
  "cover": "assets/bg/minha-capa.webp"
}

Visual Novel

O roteiro fica em um JSON dentro de data/. Cada etapa pode trocar fundo, personagens, objetos e texto.

{
  "background": "assets/bg/quarto.webp",
  "characters": [
    { "src": "assets/chars/jp.png", "position": "left" }
  ],
  "speaker": "JP",
  "mode": "pensamento",
  "text": "Linha que aparece na caixa de texto."
}

Textos

Use textos/nome-da-historia/ para a versão literária. Ela pode ser maior, com narrador, capítulos e cenas completas.

Graphic Novels

Use graphic-novels/nome-da-historia/ para páginas em quadrinhos, thumbnails, galeria ou leitura página por página.

Visual Novel no mobile

As páginas de Visual Novel abrem em modo horizontal no celular. O leitor pode tocar no botão Vertical, no canto superior, para voltar à leitura vertical. Depois disso, o mesmo botão permite retornar ao modo Horizontal.

Essa configuração fica salva no navegador do visitante e não interfere na Home nem nas páginas de texto ou graphic novel.

Estrutura pronta para crescer. A Home não precisa ser reescrita a cada história.
