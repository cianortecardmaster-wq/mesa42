# Vença o campeão nacional

Mini game interativo do Mesa 42 baseado em uma posição final de Flesh and Blood: Victor Goldmane contra Oscilio.

## Arquivos principais

- `index.html` — página da obra.
- `styles.css` — estilos do mini game.
- `script.js` — lógica do puzzle.
- `assets/` — imagens da arena, cartas e costas de carta.

Para publicar, envie a pasta `obras/venca-o-campeao-nacional/` junto com as alterações da home do Mesa 42.
